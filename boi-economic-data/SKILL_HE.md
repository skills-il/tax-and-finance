### שלב 2: שליפת נתונים מ-API של בנק ישראל
השתמשו ב-`scripts/fetch_boi_rates.py` או בקריאות API ישירות:

כתובת בסיס: `https://edge.boi.gov.il/FusionEdgeServer/sdmx/v2/data/dataflow/BOI.STATISTICS/`

מזהי סדרות מרכזיים:

| מדד | מזהה סדרה | דוגמת נתיב URL |
|------|-----------|----------------|
| ריבית בנק ישראל | ERI.SOD_ZOD.-INTEREST | `ERI_C01.H` |
| שער דולר/שקל | ER_USD_ILS | `ER?startperiod=2024-01-01&endperiod=2024-12-31` |
| שער אירו/שקל | ER_EUR_ILS | אותה תבנית כמו דולר |
| מדד המחירים (כללי) | CP.M.IL | סדרת נתוני הלמ"ס |

תבנית קריאת API:
```python
import requests

# שליפת שער חליפין יציג של בנק ישראל
url = "https://edge.boi.gov.il/FusionEdgeServer/sdmx/v2/data/dataflow/BOI.STATISTICS/ER"
params = {
    "startperiod": "2024-01-01",
    "endperiod": "2024-12-31",
    "format": "csv"
}
response = requests.get(url, params=params)
```

### שלב 3: עיבוד שערי חליפין (שער יציג)
בנק ישראל מפרסם שערי חליפין יציגים מדי יום:

| מטבע | אנגלית | קוד | שימוש אופייני |
|-------|---------|------|---------------|
| דולר אמריקאי | US Dollar | USD | מטבע חוץ ראשי, תמחור יבוא/יצוא |
| אירו | Euro | EUR | סחר עם האיחוד האירופי, נסיעות |
| לירה שטרלינג | British Pound | GBP | סחר עם בריטניה |
| ין יפני | Japanese Yen | JPY | יבוא טכנולוגיה |
| פרנק שוויצרי | Swiss Franc | CHF | מכשירים פיננסיים |

כללים מרכזיים:
- **שער יציג:** מתפרסם יומית בסביבות 15:30 שעון ישראל
- **שימושים:** חישובי מס, סילוק חוזים, חישוב מכס יבוא
- **שער יום שישי:** תקף עד מוצאי שבת וראשון (אין עדכון בסופ"ש)
- **שער חגים:** שער יום העסקים האחרון לפני החג תקף עד יום העסקים הבא

### שלב 5: חישוב מדד המחירים ואינפלציה
נתוני מדד המחירים לצרכן מהלמ"ס:

| רכיב | אנגלית | משקל (בקירוב) | הערות |
|-------|---------|---------------|-------|
| דיור | Housing | ~25% | הרכיב הגדול ביותר, כולל שכ"ד |
| מזון | Food | ~17% | תנודתי, השפעות עונתיות |
| תחבורה | Transportation | ~16% | מחירי דלק, תחבורה ציבורית |
| חינוך | Education | ~8% | שכר לימוד, ציוד לימודי |
| בריאות | Health | ~6% | השתתפויות עצמיות, תרופות |
| הלבשה | Clothing | ~4% | דפוסים עונתיים |

חישובים מבוססי מדד:
```python
# חישוב ערך מותאם אינפלציה
def adjust_for_inflation(amount, from_date, to_date, cpi_data):
    """התאמת סכום בש"ח לאינפלציה באמצעות מדד המחירים.
    
    שימושים נפוצים:
    - הסכמי שכירות (תיאום מדד)
    - מזונות
    - סעיפי הצמדה בחוזים
    - התאמת מדרגות מס
    """
    from_cpi = cpi_data[from_date]
    to_cpi = cpi_data[to_date]
    return amount * (to_cpi / from_cpi)
```

### שלב 6: הפקת דוחות
רכזו ממצאים לדוחות פעולה:
1. **סיכום ריבית:** ריבית נוכחית, תאריך שינוי אחרון, תאריך החלטה הבאה
2. **מגמות שער חליפין:** מגמות 30/90/365 יום למטבעות המבוקשים
3. **ניתוח מדד המחירים:** שיעור אינפלציה נוכחי, פירוט רכיבים, השלכות תחזית
4. **השוואה:** ריביות ישראל מול כלכלות מובילות (פד ארה"ב, ECB)
5. **הקשר:** הסבר השפעה על משכנתאות, חיסכון, עלויות עסקיות

## דוגמאות

### דוגמה 1: שליפת שער חליפין נוכחי
המשתמש אומר: "מה שער הדולר היום?"
פעולות:
1. הרצת `python scripts/fetch_boi_rates.py --rate USD`
2. החזרת שער יציג נוכחי
3. הצגת מגמת 30 יום להקשר
4. הערה: שער מתפרסם ב-15:30, לפני כן חל שער אתמול
תוצאה: שער דולר/שקל נוכחי עם הקשר מגמתי

### דוגמה 2: היסטוריית ריבית בנק ישראל
המשתמש אומר: "הראה לי שינויי ריבית בנק ישראל השנה"
פעולות:
1. הרצת `python scripts/fetch_boi_rates.py --interest --period 2024`
2. רשימת כל החלטות הריבית עם תאריכים ושינויים
3. הוספת הקשר: אינפלציה בזמן ההחלטה, סביבת ריבית עולמית
4. ציון מועד ישיבת הוועדה המוניטרית הבאה
תוצאה: ציר זמן ריבית עם הקשר כלכלי

### דוגמה 3: התאמת שכירות למדד
המשתמש אומר: "השכירות שלי 5,000 ש"ח, תתאמו למדד של השנה האחרונה"
פעולות:
1. שליפת ערכי מדד לחודש התחלה וסיום
2. חישוב: שכירות_מעודכנת = 5000 * (מדד_נוכחי / מדד_לפני_שנה)
3. הצגת: שיעור שינוי וסכום שכירות חדש
4. הערה: חוזי שכירות סטנדרטיים בישראל משתמשים ב"הצמדה למדד"
תוצאה: סכום שכירות מעודכן עם פירוט חישוב מדד

### דוגמה 4: השוואת שערים רב-מטבעית
המשתמש אומר: "השוו שערי דולר, אירו וליש"ט ב-3 החודשים האחרונים"
פעולות:
1. הרצת `python scripts/fetch_boi_rates.py --rate USD EUR GBP --period 90d`
2. הצגת שערים בטבלה עם תאריכים
3. חישוב שינויים באחוזים לתקופה
4. ציון התערבויות משמעותיות של בנק ישראל או אירועים המשפיעים על שערים
תוצאה: טבלת השוואה רב-מטבעית עם מגמות

## פתרון בעיות

### שגיאה: "BOI API rate not available"
סיבה: שאילתת שער לפני שעת הפרסום היומית (15:30 שעון ישראל) או בשבת/חג
פתרון: לפני 15:30, השתמשו בשער של יום קודם. בסופי שבוע/חגים, חל שער יום העסקים האחרון. בדקו לוח חגים ישראלי.

### שגיאה: "CPI data not yet published"
סיבה: הלמ"ס מפרסמת את המדד סביב ה-15 לחודש העוקב
פתרון: עבור החודש האחרון, ייתכן שהנתונים עדיין לא זמינים. השתמשו בנתוני חודש קודם וציינו את תאריך הפרסום הצפוי.

### שגיאה: "Exchange rate mismatch between sources"
סיבה: מקורות שונים מדווחים שערים שונים (שער בנק מסחרי מול שער יציג של בנק ישראל)
פתרון: תמיד השתמשו בשער היציג של בנק ישראל לחישובים רשמיים. שערי בנקים מסחריים כוללים מרווחים ושונים מהשער היציג.

### שגיאה: "Historical data series gap"
סיבה: ל-API של בנק ישראל עשויים להיות פערים בזמן מעברי מערכת או ארגון מחדש של נתונים
פתרון: לנתונים ישנים (לפני 2000), בדקו פרסומים סטטיסטיים של בנק ישראל. לפערים אחרונים, נסו טווחי תאריכים חלופיים או פנו לשירותי הנתונים של בנק ישראל.
