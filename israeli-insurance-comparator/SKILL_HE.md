# השוואת ביטוחים בישראל

## הוראות

### שלב 0: בניית רשימת פוליסות קיימות דרך הר הביטוח

לפני שמשווים הצעות, המשתמש צריך למשוך את רשימת הפוליסות הקיימות שלו דרך **הר הביטוח** (`https://harb.cma.gov.il/`), הפורטל של רשות שוק ההון ביטוח וחיסכון שמרכז את כל הפוליסות הפרטיות. הר הביטוח מציג כל פוליסה שהמשתמש מחזיק אצל כל מבטח ישראלי (חיים, בריאות, פנסיה, אובדן כושר, ביטוח חיים למשכנתא, ביטוח משלים בקופה, ביטוח מנהלים וכו'). הרבה משתמשים מגלים כיסויים חופפים או פוליסות שפג תוקפן ושכחו עליהן. דילוג על הצעד הזה הוא הסיבה הנפוצה ביותר שהשוואה מסתיימת בכפל ביטוחים מיותר.

הזדהות לפי תעודת זהות + קוד חד-פעמי ב-SMS או באימייל. הממשק בעברית בלבד.

### שלב 1: זיהוי סוג הביטוח הנדרש

יש לקבוע איזה מוצר ביטוחי המשתמש מעוניין להשוות. הביטוח בישראל מתחלק לקטגוריות עיקריות:

**ביטוח רכב (3 סוגים):**
- **ביטוח חובה** - חובה על פי חוק הפיצויים לנפגעי תאונות דרכים. מכסה נזקי גוף בלבד. כל רכב על הכביש חייב בביטוח זה.
- **ביטוח מקיף** - מכסה גניבה, שריפה, נזק לרכב שלך ונזק לרכוש של צד שלישי. לא חובה אך נפוץ מאוד.
- **ביטוח צד ג׳** - מכסה נזק שגרמת לרכוש של אחרים אך לא לרכב שלך. חלופה זולה יותר למקיף.

**ביטוח דירה (2 מרכיבים):**
- **מבנה** - כיסוי למבנה הבניין עצמו. נדרש לרוב על ידי הבנק למשכנתא.
- **תוכן** - כיסוי לתכולת הדירה: רהיטים, מכשירי חשמל, חפצים אישיים.

**ביטוח משלים בריאות:**
- **ביטוח משלים** - ביטוח נוסף מקופות החולים מעבר לסל הבריאות הבסיסי.

### שלב 2: איסוף פרטים נדרשים להשוואה

לפני השוואה, תאספו מהמשתמש את הפרטים הבאים:

**לביטוח רכב:**
- שם מלא ומספר תעודת זהות
- תאריך לידה ותאריך הוצאת רישיון נהיגה
- פרטי רכב: יצרן, דגם, שנת ייצור, נפח מנוע, מספר רישוי
- עיר מגורים (משפיעה על הפרמיה משמעותית)
- היסטוריית תביעות ב-3 השנים האחרונות
- מבטח נוכחי ותאריך תפוגת הפוליסה
- מספר שנות שין-נון (שנים ללא תביעות)

**לביטוח דירה:**
- כתובת הנכס וקומה
- שטח הדירה במ"ר
- שנת בנייה
- סוג (דירה, בית, פנטהאוז)
- שווי בנייה מחדש משוער (למבנה) ושווי תכולה
- האם יש משכנתא (הבנק עשוי לדרוש כיסוי ספציפי)
- אמצעי ביטחון (אזעקה, כספת, סורגים)

**לביטוח משלים בריאות:**
- גיל כל בני המשפחה שיבוטחו
- קופת חולים נוכחית (כללית, מכבי, מאוחדת, לאומית)
- דרגת השלמה נוכחית (אם קיימת)
- צרכים רפואיים ספציפיים (ניתוחים, מומחים, תרופות)

### שלב 3: שימוש בכלי השוואה ממשלתיים

התחל עם מחשבונים ממשלתיים רשמיים לקבלת מחירי בסיס אובייקטיביים:

**לביטוח חובה לרכב:**
1. היכנס ל-**car.cma.gov.il** - מחשבון הביטוח הרשמי של רשות שוק ההון (רש"ה)
2. הזן את פרטי הרכב והנהג
3. המערכת שולחת שאילתות לכל המבטחים המורשים בזמן אמת ומחזירה הצעות מחיר ממוינות
4. התוצאות מציגות: שם המבטח, פרמיה שנתית, אפשרות תשלום חודשי ופרטי כיסוי
5. הערה: תזהו המקור האמין ביותר כי חברות הביטוח מחויבות בחוק לספק הצעות מחיר מדויקות

**להשוואת ביטוח מקיף לרכב:**
1. היכנס ל-**govcarins.mof.gov.il** - סימולטור משרד האוצר
2. כלי זה עוזר להעריך עלויות ביטוח מקיף ולהבין את מרכיבי הכיסוי
3. תשוו סכומי השתתפות עצמית בין מבטחים שונים

### שלב 4: שימוש בפלטפורמות השוואה פרטיות

לאחר קבלת מחירי בסיס ממשלתיים, תבדקו בפלטפורמות פרטיות לעסקאות טובות יותר:

**Hova.co.il:**
- מתמחה בביטוח חובה לרכב
- תהליך הצעת מחיר מהיר: הזן מספר רישוי + תעודת זהות
- מציג הצעות מחיר בזמן אמת ממספר מבטחים
- ניתן לרכוש ישירות דרך הפלטפורמה

**שוקבית (shukabit.co.il):**
- פלטפורמת השוואת ביטוח מקיפה
- מכסה ביטוח רכב (כל הסוגים), דירה, בריאות ונסיעות
- מספקת טבלאות השוואה של פוליסות זו מול זו
- מציגה הבדלי כיסוי ולא רק מחיר

**Wobi (wobi.co.il):**
- אתר השוואת ביטוח ישראלי גדול (נרכש על ידי הפניקס באוקטובר 2025)
- מכסה ביטוח רכב, דירה, בריאות, חיים ועסקי
- מציע ייעוץ טלפוני עם סוכנים מורשים
- יכול לטפל בכל תהליך הרכישה
- הערה: מאז רכישת הפניקס, Wobi אינו עצמאי לחלוטין. כדאי להצליב הצעות מחיר עם פלטפורמות נוספות

**Bestie (bestie.co.il):**
- פלטפורמת השוואה עצמאית (לא חברת ביטוח ולא סוכנות)
- מספקת המלצות מותאמות אישית על בסיס שאלון קצר
- מכסה ביטוח רכב, דירה, בריאות ומשכנתא
- טוב להבנה איזו רמת כיסוי באמת נדרשת

**InsuranceFind (insurancefind.co.il):**
- מרכז הצעות מחיר ממבטחים מרובים
- טוב למוצרי ביטוח נישתיים

### שלב 5: השוואת הצעות מחיר לביטוח רכב

בעת השוואת ביטוח רכב, בנה טבלת השוואה עם העמודות הבאות:

| גורם | מבטח א׳ | מבטח ב׳ | מבטח ג׳ |
|-------|----------|----------|----------|
| פרמיה שנתית | | | |
| תשלום חודשי (אם זמין) | | | |
| השתתפות עצמית | | | |
| גרירה כלולה | | | |
| רכב חלופי | | | |
| כיסוי שמשות (ללא השתתפות עצמית) | | | |
| פוליסת חדש תמורת ישן (מגבלת גיל) | | | |
| שירותי דרך | | | |
| מוסכים מורשים לעומת בחירה חופשית | | | |

שיקולים מרכזיים:
- **הנחת שין-נון**: ככל שיש יותר שנים ללא תביעות, כך הפרמיה נמוכה יותר. ודא שמצב השין-נון שלך מדווח נכון.
- **איזון השתתפות עצמית**: השתתפות עצמית נמוכה יותר משמעה פרמיה גבוהה יותר. לנהגים טובים, השתתפות עצמית גבוהה יותר לרוב חוסכת כסף בסך הכל.
- **מוסכים מורשים**: חלק מהפוליסות דורשות שימוש ברשת המוסכים המורשים של המבטח. בחירת מוסך חופשית עולה יותר אך נותנת גמישות.
- **רכב חלופי**: תבדקו כמה ימי רכב חלופי כלולים ומה גודל הרכב.

### שלב 6: השוואת הצעות מחיר לביטוח דירה

לביטוח דירה, השווה:

| גורם | מבטח א׳ | מבטח ב׳ | מבטח ג׳ |
|-------|----------|----------|----------|
| סכום כיסוי מבנה | | | |
| סכום כיסוי תוכן | | | |
| פרמיה שנתית | | | |
| כיסוי רעידת אדמה כלול | | | |
| נזקי מים (פיצוץ צנרת) | | | |
| השתתפות עצמית בגניבה | | | |
| אחריות כלפי צד שלישי | | | |
| כיסוי אובדן שכר דירה | | | |

הערות חשובות:
- **כיסוי רעידת אדמה**: מאז 2013, פוליסות סטנדרטיות כוללות כיסוי רעידת אדמה לפי מאגר ביטוח רעידות אדמה הישראלי. ודא שזה כלול.
- **דרישת משכנתא**: אם יש לך משכנתא, הבנק בדרך כלל דורש ביטוח מבנה משועבד לטובת הבנק. שאל את המבטח לגבי שעבוד לבנק.
- **ביטוח חסר**: אם סכום הכיסוי שלך נמוך מהשווי בפועל, המבטח יכול להפעיל כלל יחסי ולהפחית כל תביעה באופן יחסי.

### שלב 7: השוואת ביטוח משלים בריאות

ביטוח משלים בריאות ישראלי פועל בדרגות מכל קופת חולים:

**כללית (כללית מושלם / כללית פלטינום):**
- מושלם: דרגת השלמה בסיסית
- פלטינום: דרגה פרימיום עם זמני המתנה קצרים יותר וגישה רחבה יותר למומחים

**מכבי (מכבי שלי / מכבי זהב):**
- מכבי שלי: השלמה בסיסית
- מכבי זהב: דרגה פרימיום

**מאוחדת (מאוחדת עדיף / מאוחדת סי):**
- עדיף: השלמה בסיסית
- סי: דרגה פרימיום

**לאומית (לאומית כסף / לאומית זהב):**
- כסף: השלמה בסיסית
- זהב: דרגה פרימיום

תשוו את הגורמים הבאים:
- פרמיה חודשית לכל בן משפחה (משתנה לפי גיל)
- ניתוח בבית חולים פרטי: אחוז כיסוי ותקרה
- ייעוצי מומחים: סכום השתתפות עצמית וזמינות
- תרופות מתקדמות שלא בסל הבריאות
- כיסוי שיניים
- כיסוי טיפולי פוריות
- חוות דעת שנייה מחו"ל
- שירותי שיקום

### שלב 8: משא ומתן ומיטוב

לאחר איסוף נתוני השוואה:

1. **פנה למבטח הנוכחי שלך** עם הצעות מחיר של מתחרים. מחלקות שימור לקוחות לעתים קרובות משתוות או מכות את מחירי המתחרים.
2. **חבילות פוליסות**: שאל על הנחות לאחזקת מספר פוליסות (רכב + דירה) אצל אותו מבטח.
3. **תזמון**: חדש או החלף 30-60 יום לפני תום הפוליסה. חידוש ברגע האחרון מקטין את כוח המשא ומתן.
4. **תשלום שנתי**: תשלום שנתי במקום חודשי חוסך לרוב 5-8%.
5. **העלאת השתתפות עצמית**: אם אתה יכול להרשות לעצמך הוצאה עצמית גבוהה יותר, העלאת ההשתתפות העצמית יכולה להפחית פרמיות משמעותית.
6. **סקירה שנתית**: שינויי חיים (רכב חדש, מעבר לאזור בטוח יותר, ילדים עזבו את הבית) יכולים להשפיע על הפרמיות.

### שלב 9: קריאה והבנת הפוליסה

לפני סיום כל רכישת ביטוח:

1. **פוליסה**: תקראו את מסמך הפוליסה המלא ולא רק את הסיכום
2. **חריגים**: תזהו מה לא מכוסה
3. **תקופת אכשרה**: חלק מהכיסויים מופעלים רק לאחר תקופת המתנה
4. **תנאי ביטול**: הבן קנסות על ביטול מוקדם
5. **תהליך תביעה**: דע כיצד להגיש תביעה ואילו מסמכים נדרשים
6. **ערוץ תלונות**: רשות שוק ההון מטפלת בתלונות צרכנים ב-pe.cma.gov.il

## דוגמאות

### דוגמה 1: רוכש ביטוח רכב לראשונה

המשתמש אומר: "קניתי Hyundai i20 שנת 2024 ואני צריך ביטוח רכב. אני גר בתל אביב, בן 28, והוצאתי רישיון בגיל 18."

פעולות:
1. הבהר שהמשתמש צריך גם ביטוח חובה וגם כנראה ביטוח מקיף לרכב חדש
2. הנחה אותו ל-car.cma.gov.il לקבלת הצעות מחיר לביטוח חובה מכל המבטחים המורשים
3. אסוף פרטי רכב: Hyundai i20 שנת 2024, נפח מנוע משוער 1.2 ליטר
4. ציין גורמים המשפיעים על הפרמיה: תל אביב (פרמיות גבוהות יותר בשל צפיפות תנועה), גיל 28 (מעבר לתוספת מתחת ל-24), 10 שנות רישיון
5. הרץ השוואות ב-Hova.co.il לחובה ואז ב-Wobi ושוקבית למקיף
6. בנה טבלת השוואה המציגה את 5 ההצעות הזולות ביותר לכל סוג
7. תמליצו לשקול: פוליסת חדש תמורת ישן (חשוב לרכב חדש), בחירת מוסך חופשית וכיסוי שמשות ללא השתתפות עצמית

תוצאה: המשתמש מקבל השוואה זו מול זו של 5 הצעות המחיר הזולות ביותר לביטוח חובה ו-5 הצעות לביטוח מקיף, עם המלצה ברורה המבוססת על יחס כיסוי-למחיר.

### דוגמה 2: ביטוח דירה לאישור משכנתא

המשתמש אומר: "הבנק אומר שאני צריך ביטוח דירה לאישור המשכנתא. הדירה שלי 85 מ"ר בחיפה, נבנתה ב-2010."

פעולות:
1. הסבר את שני המרכיבים: מבנה עבור הבנק, תוכן להגנה אישית
2. תחשבו שווי בנייה מחדש משוער בהתבסס על גודל וסוג בנייה (בדרך כלל 6,000-8,000 ש"ח למ"ר לבנייה רגילה)
3. ציין שהבנק צריך שהפוליסה תהיה משועבדת לטובתו כמוטב
4. תשוו הצעות מהראל, מגדל, כלל, הפניקס ומנורה דרך שוקבית
5. תבדקו אם לבנק למשכנתא יש מבטח מועדף (לפעמים מציע תעריפים מוזלים)
6. ודא שכיסוי רעידת אדמה כלול (חובה מאז 2013)
7. תמליצו על סכום ביטוח תוכן בהתבסס על מלאי משק בית טיפוסי

תוצאה: המשתמש מקבל 4-5 הצעות מחיר לביטוח דירה המציגות פרמיות מבנה + תוכן, עם הדרכה על הכיסוי המינימלי שהבנק יקבל וכיסוי נוסף מומלץ.

### דוגמה 3: השוואת ביטוח משלים בריאות למשפחה

המשתמש אומר: "אנחנו משפחה של 4 במכבי ושוקלים לשדרג למכבי זהב. האם זה שווה? אשתי צריכה ניתוח בברך."

פעולות:
1. תזהו דרגה נוכחית (כנראה מכבי שלי או ללא השלמה)
2. תבדקו את הטבות מכבי זהב: כיסוי ניתוח בבית חולים פרטי, גישה למומחים, זמני המתנה קצרים יותר
3. תחשבו פרמיה חודשית למשפחה (2 מבוגרים + 2 ילדים, צריך גילאים לתמחור מדויק)
4. תשוו כיסוי ניתוח: מכבי שלי לעומת מכבי זהב לניתוח אורתופדי
5. תבדקו תקופת אכשרה להטבות ניתוח - בדרך כלל 6-12 חודשים
6. תשוו עם ביטוח בריאות פרטי מחברות ביטוח מסחריות (הראל, מגדל, כלל) כחלופה
7. התחשב בדחיפות: אם הניתוח נדרש בקרוב, תקופת האכשרה עשויה להפוך את השדרוג ללא אפקטיבי בטווח הקצר

תוצאה: המשתמש מקבל ניתוח עלות-תועלת המשווה בין פרמיות מכבי שלי לזהב לעומת עלויות הניתוח הצפויות, בתוספת אפשרויות ביטוח פרטי חלופי אם תקופת האכשרה מהווה בעיה.

## משבר ביטוח סיעודי 2025-2026

שינוי מבני גדול שיבש את שוק הביטוח הסיעודי ב-2025-2026, וכל השוואה ב-2026 חייבת להציף אותו:

- **משרד הבריאות הורה לארבע קופות החולים להפסיק למכור פוליסות ביטוח סיעודי חדשות מ-1.7.2025**, ולסיים את כל פעילות הביטוח הסיעודי עד 1.1.2026. הפוליסות בערוץ הקופות שמיליוני ישראלים החזיקו אינן עוד המודל המומלץ.
- **רשות שוק ההון הקשיחה את הגדרת זכאות החולה ל-4 מתוך 6 ADL** (פעולות יומיומיות) מ-1.1.2025 (היה 3 מתוך 6). חולים שהיו זכאים בהגדרה הישנה עשויים לא להיות זכאים עכשיו.
- **אפשרויות פתוחות לכיסוי סיעודי חדש:** ביטוח סיעודי פרטי מהמבטחים הגדולים (פניקס, מגדל, הראל, כלל, מנורה), ומסלול הסיעוד של ביטוח לאומי. תמחור וחיתום משתנים משמעותית. אמתו זמינות נוכחית מול כל מבטח, השוק עדיין במצב של התאמה.

כשמשתמש שואל על "ביטוח סיעודי" ב-2026, הציפו את השיבוש במפורש, ציטוט מסגרת ערוץ הקופות מלפני 2025 כאילו עדיין פעיל יוצר רושם מטעה.

## הקשר ביטוחי לחרבות ברזל / מילואים

- **פוליסות משרתי מילואים:** גיוס בצו 8 עשוי להפעיל התאמות אוטומטיות באובדן כושר ובביטוח חיים (חלק מהמבטחים מוותרים על פרמיות; אחרים מפעילים חרגים סטנדרטיים). מסלול הסיעוד של ביטוח לאומי פועל בנפרד ומשלם על נכות או מוות הקשורים בשירות.
- **חרגים בביטוח נסיעות לחו"ל:** מאז חרבות ברזל (אוקטובר 2023) רוב מבטחי הנסיעות מחריגים יעדים באזורי מלחמה פעילים ועלולים להחריג טיסות שעוברות דרכם. אמתו את סעיף החרגה הגיאוגרפי לפני רכישה.
- **חרגי טרור ומלחמה בביטוח חיים ואובדן כושר:** בדקו את ההגדרות של "סיכון מלחמה" ו"טרור" בפוליסה; חלק מהתשלומים מוקטנים או מוחרגים בפעולות איבה מאורגנות, אירועי טרור טהורים בדרך כלל מכוסים (אמתו לפי פוליסה).

## מלכודות נפוצות
- לביטוח רכב בישראל שלוש שכבות חובה/רשות נפרדות: חובה (צד שלישי חובה), מקיף (comprehensive), וצד ג' (רכוש צד שלישי). סוכנים עלולים לערבב ביניהן או להשתמש במונחי ביטוח אמריקאיים/בריטיים שלא מתאימים לקטגוריות ישראליות.
- פרמיות ביטוח בישראל מצוטטות לשנה, לא לחודש. סוכנים עלולים להציג מחירים חודשיים כשה-API מחזיר נתונים שנתיים, מה שגורם לבלבול של פי 12.
- ביטוח חובה (רכב) מוסדר על ידי המדינה והמחירים נקבעים על ידי ועדה. סוכנים עלולים להציע "לחפש הצעות" לסוג זה, שבו השוני במחירים מוגבל.
- ביטוח בריאות בישראל כולל שכבה ציבורית (קופת חולים) ושכבות פרטיות משלימות (שב"ן, משלים). סוכנים עלולים להשוות רק תוכניות פרטיות בלי לציין את הכיסוי הציבורי האוניברסלי.

## פתרון בעיות

### שגיאה: "מחשבון רשות שוק ההון ב-car.cma.gov.il לא מחזיר תוצאות"

סיבה: מחשבון רש"ה דורש פרטי רישום רכב מדויקים. בעיות נפוצות כוללות פורמט מספר רישוי שגוי, רכב שעדיין לא רשום במאגר משרד התחבורה (לרכבים חדשים מאוד), או שהמחשבון מושבת זמנית במהלך שעות תחזוקה (בדרך כלל בלילה).

פתרון: ודא את פורמט מספר הרישוי (7 או 8 ספרות, ללא מקפים). לרכבים שנרכשו לאחרונה, המתן 24-48 שעות לאחר הרישום לפני ניסיון במחשבון. אם האתר מושבת, נסה בשעות העבודה (ראשון-חמישי 8:00-17:00). כחלופה, תשתמשו ב-Hova.co.il שמתחזק חיבורי API עצמאיים למבטחים.

### שגיאה: "הצעות מחיר לביטוח משתנות באופן קיצוני בין פלטפורמות עבור אותם פרטים"

סיבה: פלטפורמות שונות עשויות להשתמש במודלי חיתום שונים מעט, להחזיק בעסקאות בלעדיות עם מבטחים מסוימים, או להחיל הנחות קידום מכירות שלא זמינות במקומות אחרים. חלק מהפלטפורמות גם מציגות מחיר בסיס לפני מיסים ועמלות בעוד שאחרות מציגות את המחיר הסופי.

פתרון: תשוו תמיד את הפרמיה השנתית הסופית כולל כל העמלות והמיסים (מע"מ כלול בפרמיות ביטוח בישראל). תבדקו אם ההצעה כוללת "רישיון שימוש" שחלק מהפלטפורמות רושמות בנפרד. תשתמשו במחשבון הממשלתי של רש"ה כבסיס כי הוא מציג תמחור סטנדרטי, ואז תבדקו אם פלטפורמות פרטיות מציעות עסקאות טובות יותר. כשיש ספק, תתקשרו ישירות למבטח לאימות המחיר המצוטט.

### שגיאה: "שנות השין-נון שלי לא תואמות למה שהמבטח מציג"

סיבה: רישום השין-נון מנוהל על ידי הפול (מאגר הביטוח הישראלי). מבטחים שולחים שאילתות למאגר זה, אך לפעמים יש עיכובים בעדכון, במיוחד אם החלפת מבטח לאחרונה או הייתה תביעה שנמשכה מאוחר יותר או הוסדרה כתביעה שלא באשמתך.

פתרון: תבקשו את רישום השין-נון הרשמי שלך מהמבטח הנוכחי או הקודם. ניתן גם לפנות ישירות לפול. אם יש טעות, הגש בקשת תיקון דרך המבטח שלך. תשמרו עותקים של כל אישורי העדר תביעות מפוליסות קודמות. ההבדל בין רמות שין-נון יכול להיות 10-20% בפרמיות, כך ששווה לתקן כל טעות.

### שגיאה: "פלטפורמת ההשוואה מבקשת מספר ת.ז. אבל לא נוח לי לשתף"

סיבה: הצעות מחיר לביטוח ישראלי דורשות מספר תעודת זהות לצרכי ציות רגולטורי ולשאילתת מאגר השין-נון ומרשם הרכב.

פתרון: פלטפורמות ממשלתיות (car.cma.gov.il) מאובטחות ומפוקחות. לפלטפורמות פרטיות, ודא שהן סוכני ביטוח או ברוקרים מורשים הרשומים באתר רש"ה. ניתן להתחיל עם פלטפורמות שמספקות הערכות ראשוניות ללא ת.ז. (כמו המחשבון הכללי של Wobi) ולהזין את הת.ז. רק בפלטפורמה שבחרת לרכישה. לעולם אל תשתף את מספר הת.ז. באתרים לא מאובטחים או באימייל.
